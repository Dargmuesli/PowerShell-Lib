# Convert-PSObjectToHashtable

## SYNOPSIS
Converts a PSCustomObject to a hashtable.

## SYNTAX

```
Convert-PSObjectToHashtable [-InputObject] <PSObject> [<CommonParameters>]
```

## DESCRIPTION
DEPRECATED function.

## EXAMPLES

### -------------------------- BEISPIEL 1 --------------------------
```
Convert-PSObjectToHashtable -InputObject $InputObject
```
