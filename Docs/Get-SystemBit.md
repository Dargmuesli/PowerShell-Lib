# Get-SystemBit

## SYNOPSIS
Gets the processor bit architecture.

## SYNTAX

```
Get-SystemBit
```

## DESCRIPTION
The "Get-SystemBit" cmdlet gets the size of "IntPtr" and return either 32 or 64 accordingly.
