# Test-DockerInSwarm

## SYNOPSIS
Checks if Docker is in swarm-mode.

## SYNTAX

```
Test-DockerInSwarm [<CommonParameters>]
```

## DESCRIPTION
The "Test-DockerInSwarm" cmdlet tries to create a swarm and returns false if successful, leaving the just created swarm.

## EXAMPLES

### -------------------------- BEISPIEL 1 --------------------------
```
Test-DockerInSwarm
```
