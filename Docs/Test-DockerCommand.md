# Test-DockerCommand

## SYNOPSIS
Checks the availability of the "docker" command.

## SYNTAX

```
Test-DockerCommand [<CommonParameters>]
```

## DESCRIPTION
The "Test-DockerCommand" cmdlet tries to get the "docker" command and returns true on success.

## EXAMPLES

### -------------------------- BEISPIEL 1 --------------------------
```
Test-DockerCommand
```
