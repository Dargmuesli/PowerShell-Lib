# Install-Docker

## SYNOPSIS
Installs Docker.

## SYNTAX

```
Install-Docker [[-Edition] <String>] [[-DownloadMethod] <String>] [<CommonParameters>]
```

## DESCRIPTION
The "Install-Docker" cmdlet downloads and executes the Docker installer.

## EXAMPLES

### -------------------------- BEISPIEL 1 --------------------------
```
Install-Docker
```
