# Test-DockerRunning

## SYNOPSIS
Checks if Docker is running.

## SYNTAX

```
Test-DockerRunning [<CommonParameters>]
```

## DESCRIPTION
The "Test-DockerRunning" cmdlet tries to verify the availability of the "docker ps" command and returns true on success.

## EXAMPLES

### -------------------------- BEISPIEL 1 --------------------------
```
Test-DockerRunning
```
