# Test-IsMacOS

## SYNOPSIS
Tests if the current platform is MacOS.

## SYNTAX

```
Test-IsMacOS
```

## DESCRIPTION
The "Test-IsMacOS" cmdlet checks whether the "IsMacOS" variable is set and returns its value.
