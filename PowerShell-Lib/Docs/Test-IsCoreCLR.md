# Test-IsCoreCLR

## SYNOPSIS
Tests if PowerShell Core is executed.

## SYNTAX

```
Test-IsCoreCLR
```

## DESCRIPTION
The "Test-IsCoreCLR" cmdlet checks whether the PS version table contains the "PSEdition" key having the value "Core".
