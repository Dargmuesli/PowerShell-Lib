# Test-IsLinux

## SYNOPSIS
Tests if the current platform is Linux.

## SYNTAX

```
Test-IsLinux
```

## DESCRIPTION
The "Test-IsLinux" cmdlet checks whether the "IsLinux" variable is set and returns its value.

## EXAMPLES

### EXAMPLE 1
