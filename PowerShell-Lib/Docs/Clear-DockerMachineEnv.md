# Clear-DockerMachineEnv

## SYNOPSIS
Removes all Docker Machine environment variables.

## SYNTAX

```
Clear-DockerMachineEnv
```

## DESCRIPTION
The "Clear-DockerMachineEnv" cmdlet unsets the Docker Machine environment variables.

## EXAMPLES

### BEISPIEL 1
