# Test-IsWindows

## SYNOPSIS
Tests if the current platform is Windows.

## SYNTAX

```
Test-IsWindows
```

## DESCRIPTION
The "Test-IsWindows" cmdlet checks whether the "IsWindows" variable is set and returns its value.

### BEISPIEL 1
