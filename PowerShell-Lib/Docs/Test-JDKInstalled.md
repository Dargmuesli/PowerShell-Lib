# Test-JDKInstalled

## SYNOPSIS
Checks whether a JDK is installed.

## SYNTAX

```
Test-JDKInstalled
```

## DESCRIPTION
The "Test-JDKInstalled" cmdlet tests if an app with a name matching a regular expression for valid JDK names is installed and returns true if that's the case.

## EXAMPLES

### BEISPIEL 1
```
Test-JDKInstalled
```
