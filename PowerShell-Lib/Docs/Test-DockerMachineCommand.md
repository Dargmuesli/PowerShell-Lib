# Test-DockerMachineCommand

## SYNOPSIS
Checks the availability of the "docker-machine" command.

## SYNTAX

```
Test-DockerMachineCommand
```

## DESCRIPTION
The "Test-DockerMachineCommand" cmdlet tries to get the "docker-machine" command and returns true on success.
