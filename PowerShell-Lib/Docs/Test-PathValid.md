# Test-PathValid

## SYNOPSIS
Checks whether a path is valid.

## SYNTAX

```
Test-PathValid [-Path] <Object>
```

## DESCRIPTION
The "Test-PathValid" cmdlet checks if a path is valid and returns true on success.

## EXAMPLES

### -------------------------- BEISPIEL 1 --------------------------
```
Test-PathValid -Path ".\"
```

## PARAMETERS

### -Path
The path that is to be checked.
