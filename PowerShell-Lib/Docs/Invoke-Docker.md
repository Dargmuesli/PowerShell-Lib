# Invoke-Docker

## SYNOPSIS
Invokes docker commands.

## SYNTAX

```
Invoke-Docker
```

## DESCRIPTION
The "Invoke-Docker" cmdlet verifies the existence of the "docker" command and executes it with the given parameters.

## EXAMPLES

### BEISPIEL 1
```
Invoke-Docker start registry
```
