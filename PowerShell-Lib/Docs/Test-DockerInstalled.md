# Test-DockerInstalled

## SYNOPSIS
Checks whether Docker is installed.

## SYNTAX

```
Test-DockerInstalled
```

## DESCRIPTION
The "Test-DockerInstalled" cmdlet tries to access the command "docker" and "docker-machine" and returns true if at least one is successful.

### -------------------------- BEISPIEL 1 --------------------------
```
Test-DockerInstalled
```
