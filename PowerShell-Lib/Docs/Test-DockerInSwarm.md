# Test-DockerInSwarm

## SYNOPSIS
Checks whether Docker is in swarm-mode.

## SYNTAX

```
Test-DockerInSwarm
```

## DESCRIPTION
The "Test-DockerInSwarm" cmdlet tries to create a swarm and returns false if successful, leaving the just created swarm.

## EXAMPLES

### EXAMPLE 1
```
Test-DockerInSwarm
```
