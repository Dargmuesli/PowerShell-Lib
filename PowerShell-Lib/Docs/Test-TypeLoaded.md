# Test-TypeLoaded

## SYNOPSIS
Checks whether a type is loaded.

## SYNTAX

```
Test-TypeLoaded [-Name] <String> [<CommonParameters>]
```

## DESCRIPTION
The "Test-TypeLoaded" cmdlet .

## EXAMPLES

### EXAMPLE 1
```
Test-TypeLoaded -Name "YamlDotNet"
```
