# Test-JREInstalled

## SYNOPSIS
Checks whether a JRE is installed.

## SYNTAX

```
Test-JREInstalled
```

## DESCRIPTION
The "Test-JREInstalled" cmdlet tests if an app with a name matching a regular expression for valid JRE names is installed and returns true if that's the case.

## EXAMPLES

### -------------------------- BEISPIEL 1 --------------------------
```
Test-JREInstalled
```
