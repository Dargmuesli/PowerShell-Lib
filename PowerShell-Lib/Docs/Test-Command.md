# Test-Command

## SYNOPSIS
Tests a command.

## SYNTAX

```
Test-Command [-Command] <String>
```

## DESCRIPTION
The "Test-Command" cmdlet checks if a command is available and returns true on success.

## EXAMPLES

### -------------------------- BEISPIEL 1 --------------------------
```
Test-Command -Command "docker"
```

## PARAMETERS

### -Command
The command that is to be checked.
