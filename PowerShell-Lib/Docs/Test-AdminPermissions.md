# Test-AdminPermissions

## SYNOPSIS
Tests if administrator permissions are enabled.

## SYNTAX

```
Test-AdminPermissions
```

## DESCRIPTION
The "Test-AdminPermissions" cmdlet verifies that administrator permissions are enabled and returns true on success.
