# Get-LinuxOsId

## SYNOPSIS
Returns the Linux OS ID.

## SYNTAX

```
Get-LinuxOsId
```

## DESCRIPTION
The "Get-LinuxOsId" cmdlet mounts the "/etc/os-release" file and returns the operating system's ID.

## EXAMPLES

### EXAMPLE 1
```
Get-LinuxOsId
```
