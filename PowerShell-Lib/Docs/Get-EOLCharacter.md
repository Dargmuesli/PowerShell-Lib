# Get-EOLCharacter

## SYNOPSIS
Returns the current platform's EOL character.

## SYNTAX

```
Get-EOLCharacter
```

## DESCRIPTION
The "Get-EOLCharacter" cmdlet returns the current platform's end of line character.

## EXAMPLES

### BEISPIEL 1
```
Get-EOLCharacter
```
