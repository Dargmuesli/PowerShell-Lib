# Test-DockerForWinInstalled

## SYNOPSIS
Checks whether Docker for Windows is installed.

## SYNTAX

```
Test-DockerForWinInstalled
```

## DESCRIPTION
The "Test-DockerForWinInstalled" cmdlet tests if an app with the name "Docker" is installed and returns true if that's the case.

### -------------------------- BEISPIEL 1 --------------------------
```
Test-DockerForWinInstalled
```
